<h3>Objetivo Geral</h3>

O objetivo geral deste projeto é desenvolver um jogo de plataforma 2D utilizando a engine Godot 4.6, integrado a um banco de dados relacional via API, para a disciplina de Análise Orientada a Objetos (AOO). O Game Lambda visa demonstrar a aplicação prática de conceitos de programação orientada a objetos (como herança, polimorfismo e encapsulamento) na estruturação das entidades do jogo (herói, inimigos, itens, mercador). O jogo conta com um ciclo de progressão de 4 cenários, envolvendo coleta de recursos, compra de buffs (armadura e botas), um combate contra um chefe final e uma mecânica assíncrona onde o último cenário interage com o banco de dados para nomear os itens da loja com base no último jogador derrotado.

<hr>

<h3>Ferramentas e tecnologias utilizadas</h3>

- Front-End / Engine: Godot 4.6 (GDScript)

- Back-End (API): Python (Flask / FastAPI)

- Banco de Dados: MySQL

- Versionamento: Git e GitHub
